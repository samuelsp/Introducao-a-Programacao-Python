Exibindo Texto em Python

Python possui recursos para exibição de texto no terminal. É possível formatar o texto para mostrar ao usuário informações de forma mais conveniente.




Quase todo programa utiliza texto para informar ao usuário o que esta ocorrendo. O texto é um dos meios de interação mais utilizados por programadores. Por isso é importante utilizar uma linguagem clara e objetiva além dos recursos disponíveis para formatação de texto.

Utilizando o print


  print("Gut! Ficou fantastico este texto formatado com a função print!")
            

                     print("Veja como utilizar o print")

É necessário utilizar aspas simples(') ou duplas(") dentro do comando print para indicar que o conteúdo é uma cadeia de caracteres ou string. Uma string é qualquer texto.


E por que existe as aspas simples e as aspas duplas?



                print("Você pode usar palavras com apóstrofo.")
                print("It's an object!")

Dúvida?

Legal e se meu texto não couber em uma linha eu terei que usar varios comandos print?

A resposta é não panda você poderá usar apenas um print e substituir as aspas simples ou duplas pela triplo aspas simples ou dupla. Mas como?

Assim:


                    print('''Aqui você insere um texto que não irá
                           caber em apenas uma linha''')

Ou assim:


                    print("""Aqui você insere um texto que não irá
                          caber em apenas uma linha""")

Formatações com print


                \n - pula uma linha ou insere nova linha
                \t - insere tabulação ou insere um espaço entre palavras
                + - concatena strings
                \ - caractere de escape

Utilidade do caractere de escape


    print("Eu posso inserir uma palavra que contém \"aspas\" em meu texto.")
    print('Eu posso inserir uma palavra que contém \'aspas\' em meu texto.')
    print("C:\\Windows\\Documents\\file.txt")
